### What This Is
An interactive game night platform where **hosts** run live game shows and **viewers** actively participate in real-time using their phones as controllers. Think HQ Trivia meets Twitch meets Jackbox Party Pack.

### Core Concept
- Hosts stream live (via Twitch/YouTube embed)
- Viewers join games and compete/collaborate in real-time
- Game types: Trivia, Drawing challenges, Physical challenges, Team puzzles
- Live leaderboards with real prizes (sponsored or host-funded)
- Mobile-first viewer experience (phone as game controller)
